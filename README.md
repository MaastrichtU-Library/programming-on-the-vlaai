# Programming on the VLAAI

Minimal Quarto website MVP.

## Pages

- `index.qmd`: overview page
- `course.qmd`: dummy course content page (temporary not displayed)
- `about.qmd` : Overview of the Programming Café, including the CAFE method it is based on and links to other Programming Cafés in the Netherlands. (temporarily not displayed)
## Render locally

```sh
quarto render
```

The rendered website is written to `_site/`.
